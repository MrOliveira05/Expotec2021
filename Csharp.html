    C# é uma linguagem de programação criada pela Microsoft como parte da plataforma .NET. Por ter uma sintaxe parecida com o Java e C++, o C# pode ser indicado mesmo para programadores iniciantes, que já conhecem a programação Orientada a Objetos. Com ela podemos criar desde aplicações web, desktop, mobile e também jogos eletrônicos.



    Ferramentas

    Um site que eu recomendo pra programar c# é o Replit.
    Mas caso você tenha um computador bom, eu recomendo usar o Visual Studio.

  • String: é utilizado para se armazenar caracteres.
  • Double: número fracionado ou em decimal.
  • Bool: Verdadeiro ou Falso.

    Escrita

  • Console.Write: Escreve algo na tela para o usuário.
  • Console.ReadLine: Ele lê a linha que o usuário digitou e guarda numa variável.
  • Convert.ToInt32: Ele converte o número que o usuário digitou em numero inteiro de 32bits.
  • Convert.ToDouble: Ele converte o número que o usuário digitou em numero decimal ou fracionado.
  • Console.ReadKey: Ele espera o usuário digitar algo para fechar o console(prompt).














    • if e else

  • if: Serve para verificar se a condição está certa.

    if(a<=10)
    {
       Console.Writeline("É Verdadeiro");
    }

  • else: Se a condição do if não estiver correta, o else irá ficar com o resto.

    else
    {
       Console.Writeline("É falso");
    }

























    • Estruturas de Repetições

  • For: Ele repete até chegar no resultado final.
    for (int i = 0; i < 3; i++)
    {
    Console.Write(i);
    }
  • While: É a mesma coisa do for, mas para funcionar temos que declarar o valor antes.
    int n = 0;
    while (n < 5)
    {
    Console.Write(n);
    n++;
    }
  • Do...While: Ele só vai repetir enquanto a condição for atendida.
    int n = 0;
    do
    {
    Console.Write(n);
    n++;
    } while (n < 5);

























    • Escolha

  • Switch Case: é um comando para fazer escolhas(menus).
    Sintaxe:
    switch (variável)
    {
       case 1:
       Instruções;
       break;

       case 2:
       Instruções;
       break;

       default:
       Instruções;
    }
