    • Tags de PHP


    O PHP (um acrônimo recursivo para PHP: Hypertext Preprocessor ) é uma linguagem de script open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML.



    Programa

    O php não é igual o HTML, que é só abrir o arquivo que já funciona.
    Ele precisa de um servidor para poder funcionar, e pra isso precisamos de um aplicativo chamado XAMPP.
    Para podermos abrir um arquivo php, precisamos salvar o arquivo no C:/xampp/htdocs, ativar o apache no xampp, e no navegador colocar o seguinte código:
    //localhost:80, 85 ou 8080(depende da sua porta no apache)/nome do seu arquivo
    E pronto, o seu arquivo irá abrir no seu navegador.














    Começo

    Para começar a programar, precisamos abrir uma tag em PHP, e essa tag é <?php
    ?>
    Tudo que formos programar em php precisamos colocar dentro dessa tag.










    Variáveis

    As variáveis no PHP são representadas por um cifrão ($) seguido pelo nome da variável. Os nomes de variável são case-sensitive.
    Nomes de variável seguem as mesmas regras como outros rótulos no PHP. Um nome de variável válido inicia-se com uma letra ou sublinhado, seguido de qualquer número de letras, números ou sublinhados.

    Nota: Para nosso propósito, as letras a-z, A-Z e os bytes de 127 a 255 (0x7f-0xff).
    Nota: $this é uma variável especial que não pode ser atribuída.














    Echo

    A tag echo serve para imprimir algo na tela.
    Sintaxe: echo"Olá mundo";

    if else

    if: Serve para verificar se a condição está certa.

    if(($ano % 4 == 0 && $ano % 100!=0) || $ano % 400==0)
    {
       echo "o ano ".$ano. " é bissexto";
    }

  • else: Se a condição do if não estiver correta, o else irá ficar com o resto.

    else
    {
       echo "O ano ".$ano." não é bissexto";
    }
