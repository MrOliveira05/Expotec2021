    • Tags de HTML5


    O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador da Web. Um elemento HTML é separado de outro texto em um documento por "tags", que consistem no nome do elemento entre "<" e ">". O nome de um elemento dentro de uma tag é insensível a maiúsculas e minúsculas. Isto é, pode ser escrito em maiúsculas, minúsculas ou um mistura.



    <!DOCTYPE html>

    O <!DOCTYPE html> não é uma tag, porém é uma declaração obrigatória em um código HTML. Essa declaração informa o navegador a versão de HTML utilizada no arquivo.
